# Graphics project
This is my project for the graphics module at the university of nottingham. It contains a scene of a street with generated terrain and dynamic lighting.




## Controls

wasd - Standard movement

ijkl - Remote Control Car Movement

Space - Fly up

Ctrl - Fly down

Click + drag - Change camera Orientation

1 - Change to floating camera

2 - Change to static camera (on RC car)

Esc - Close program
